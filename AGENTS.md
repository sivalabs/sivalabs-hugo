# AGENTS.md

This file provides guidance to AI Agents when working with code in this repository.

## What this is

Personal blog for [sivalabs.in](https://www.sivalabs.in) — a Hugo static site using the custom `sivalabs` theme, deployed to Netlify.

## Commands

```shell
npm install          # install dependencies (imagemin for image optimization)
npm run dev          # optimize images then serve with drafts/future posts enabled
npm run build        # production build → build/
npm run serve        # serve only (skip image optimization)
npm run clean        # rm -rf build resources public
npm run images:optimize  # convert static/raw-images/*.{jpg,jpeg,png} → .webp
task                 # alias for npm run dev
```

Hugo version: **0.147.9** (set in `netlify.toml`).

## Content structure

```
content/
  blog/<year>/<slug>.md     # blog posts (flat .md files, NOT leaf bundles)
  pages/*.md                # static pages (about, tutorials, etc.)
  trainings/*.md            # training course pages
static/
  raw-images/               # source images — add originals here
  raw-images/                   # full-size webp (generated by images:optimize)
  raw-images/           # 600×340 webp thumbnails (generated)
```

## Blog post frontmatter

```yaml
---
title: "Post Title"
author: Siva
images: ["/raw-images/<slug>.webp"]   # og:image / social card
type: post
draft: false
date: 2026-01-01T00:00:00+05:30
url: /blog/<slug>
toc: true                          # show table of contents
categories: [SpringBoot]
tags: [Java, SpringBoot]
---
```

Posts use `<!--more-->` to set the excerpt boundary.

## Image workflow

1. Drop source image in `static/raw-images/` as `.jpg`, `.jpeg`, or `.png`.
2. Run `npm run images:optimize` — it writes `.webp` to both `static/raw-images/` (full size) and `static/raw-images/` (600×340).
3. Reference in frontmatter as `images: ["/raw-images/<name>.webp"]`.

## Deployment

Netlify auto-deploys on push to `main`. Build command: `hugo --gc --minify`. Output dir: `public/`.
